# tldr
Homemade TL;DR for the man page

# install
```bash
$ git clone [repo]
# add a config
$ echo 'DATADIR=path/to/data' > ~/.tldr.cfg
# optional: make an alias
$ alias tldr=bash path/to/tldr.sh
```

# usage
```bash
$ tldr <cmd>
```
